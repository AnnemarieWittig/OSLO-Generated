Participatie Rol

Beschrijving
Beschrijft de parcipatie van verschillende Agents in de context van de consumptie van een bepaalde Publieke Dienstverlening.
Label Definitie Notitie Notatie
Aanvrager Agent die de subsidiemaatregel of publieke dienstverlening aanvraagt. Aanvrager
Begunstigde Agent die de begunstigde is van de maatregel of publieke dienstverlening, bv. de agent die finaal het financieel voordeel geniet. Begunstigde
Beslisser De agent die een beslissing neemt over het al dan niet goedkeuren van een bepaalde aanvraag, het toekennen en/of het uitbetalen van een bepaald subsidiebedrag. Beslisser
Betrokkene Een betrokkene is een Agent wie het rechtstreeks aangaat, zonder dat diens belang daarbij een rol hoeft te spelen of zelf actief deelneemt aan de dienstverlening. Betrokkene
Expert Een (extern) expert kan worden aangesteld om een bepaalde aanvraag, bijvoorbeeld in het kader van een subsidiedossier, inhoudelijk te beoordelen. Expert
Indiener Agent die de aanvraag officieel indient bij de bevoegde instantie. Indiener
