REST API richtlijnen voor Informatie Vlaanderen

Voor het bouwen van RESTful API's binnen Informatie Vlaanderen zijn richtlijnen opgesteld. Deze richtlijnen zijn van toepassing op nieuwe ontwikkelingen en bij grote herwerkingen van bestaande oplossingen. Deze richtlijnen zijn het resultaat van een interne afstemming via de Expertise Groep Architectuur en werden afgetoetst met externe partijen die gelijkaardige afspraken maken.


Het volledige document kan je hier terugvinden.


Vlaamse URI-Standaard Validator

De Vlaamse URI-Standaard Validator is een tool die het mogelijk maakt om na te gaan of URI's die door Vlaamse overheden worden toegekend voldoen aan de vormregels om data-resources op een uniforme manier te onsluiten via het Web. Een gebruiker kan een URI meegeven aan de validator om te controleren hoe goed zijn URI aan de vormregels voldoet. De gebruiker krijgt feedback terug op welke vlakken zijn URI wel of niet voldoet aan de regels.

Informatie Vlaanderen JSON-LD Playground

Informatie Vlaanderen heeft zijn eigen JSON-LD Playground ontwikkeld. Voor elk bestaand applicatieprofiel is er een voorbeeld gedefinieerd zodat mensen van daar kunnen starten. Meer ervaren gebruikers kunnen zelf aan de slag gaan met hun JSON-LD data. Gebruikers hebben ook de mogelijkheid om hun input te laten valideren door de OSLO Validator.
